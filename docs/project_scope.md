# Projektumfang

## Ziel des Projekts

Ein adaptiver Agent passt die Schwierigkeit einer Spielsimulation an die Leistung eines simulierten Spielers an.

## Bestandteil der ersten Version

* drei Spielertypen:
  * Anfänger
  * durchschnittlicher Spieler
  * erfahrener Spieler
* Schwierigkeitsstufen von 1 bis 10
* drei veränderbare Spielparameter:
  * Gegneranzahl
  * Gegnerschaden
  * Wahrscheinlichkeit für Heilungsobjekte
* Ergebnisse einer Spielrunde:
  * Sieg oder Niederlage
  * verbleibende Lebenspunkte
  * erlittener Schaden
  * Trefferquote
  * Rundendauer
* Vergleich zwischen:
  * statischem Schwierigkeitssystem
  * adaptivem Schwierigkeitssystem
* Zielgewinnrate von 60 Prozent
* regelbasierter Difficulty Agent
* mindestens drei Ergebnisdiagramme

## Nicht Bestandteil der ersten Version

* echte Game Engine
* Reinforcement Learning
* LLM als entscheidende Steuerungsinstanz
* Weboberfläche
* reale Spielerdaten
* komplexe Gegner-KI
* Multiplayer-Funktionen

## Forschungsfrage

„Kann ein adaptiver Schwierigkeitsagent eine ausgewogenere Spielerfahrung erzeugen als ein statisches Schwierigkeitssystem?“

## Erfolgskriterien

Die erste Version gilt als erfolgreich, wenn:

* alle drei Spielertypen simuliert werden können,
* mehrere Spielrunden durchgeführt werden können,
* der Agent die Schwierigkeit erhöhen, reduzieren oder beibehalten kann,
* statisches und adaptives System vergleichbar sind,
* Ergebnisse gespeichert und grafisch dargestellt werden können.
